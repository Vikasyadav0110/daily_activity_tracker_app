## Critical Retention Moments (Design Must-Haves)

| Moment | Trigger | Design Response |
|---|---|---|
| Day 1: First check-off | Tap | < 100ms response + haptic + streak=1 animation |
| Day 4–5: First miss | No log by midnight | "48h grace" push notification at 8 AM next day |
| Day 7: First week | 7th check-off | Badge unlock animation + WhatsApp share card |
| Day 14: D14 churn risk | 2-week mark | Summary card + new goal prompt |
| Any miss > 48h | Streak reset | "New start" framing — not failure language |

---

## Pain Points to Avoid

- ❌ Slow check-off (> 100ms) = breaks flow state during study sessions
- ❌ Overly complex onboarding (> 3 steps) = Raj drops out before first value
- ❌ English-only app = Raj doesn't recommend it to Hindi-medium friends
- ❌ Hard streak reset language ("You failed!") = shame → uninstall
- ❌ APK > 15 MB = Raj doesn't install on his low-storage secondary phone
